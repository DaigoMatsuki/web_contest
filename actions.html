私たちにできること

✨ 問題を知ったら、次は行動！今日からできることがあります

ここまで読んで、「じゃあ、自分に何ができるの？」と思ったかもしれません。安心してください。誰にでもできる小さな一歩があります。

🙋 消費者（私たち）にできること

服との付き合い方を、3つのステップで見直してみましょう。

🛒 STEP 1：買うとき

✓ 「本当に必要？」と自分に問いかける

安いからと衝動買いせず、「1年後も着ているか」を想像してみましょう。

✓ 長く使える品質を選ぶ

「1,000円で3回着る服」より「3,000円で30回着る服」の方が、環境にも経済的にも優しいです。

✓ 修繕しやすいデザインを選ぶ

ボタンやファスナーが直せる、シンプルなデザインは長持ちします。

👕 STEP 2：使うとき

✓ 洗濯表示を守る

正しい洗い方をするだけで、服の寿命は大きく変わります。

✓ 破れたら直してみる

ボタンが取れたり、少し破れたりしても、簡単な修理で復活します。YouTube などで直し方を調べてみましょう。

✓ 「次の人」を意識する

いつか手放すことを考えて、きれいに扱いましょう。

♻️ STEP 3：手放すとき

✓ 状態をチェックする

「本当にまだ着られるか？」汚れ、破れ、臭いがないか確認しましょう。使えない状態なら、寄付ではなく適切に廃棄を。

✓ 回収先を調べる

回収ボックスに入れる前に、その団体のウェブサイトで「回収した服がどこへ行くのか」を確認してみましょう。

✓ 様々な選択肢を知る

・リサイクルショップ（地域の店で直接売る）
・フリマアプリ（メルカリ、ラクマなど）
・友人や家族に譲る
・地域の支援団体に寄付（直接渡せる場所がベスト）

🏢 企業にできること・求められること

📐 設計段階

・リサイクル可能な素材を選ぶ
・長く使えるデザインにする
・修理しやすい構造にする

♻️ 回収・処理

・回収した服の行き先を公開
・適切な選別と処理
・輸出先の状況を把握する

💡 企業に求めたいこと：

「作って終わり」ではなく、最後まで責任を持つ姿勢。そして、消費者に対して正直な情報を伝えること。

🚛 回収者・業者にできること・求められること

  • 選別の徹底：現地で再利用可能なものとそうでないものを明確に分ける
  • 輸出量の管理：現地の処理能力を超えない運用を検討する
  • 情報開示：回収した服がどこへ行くのか、透明性を持って伝える
  • 現地との対話：受け入れ先の状況や要望を把握する

🌏 すべての立場で共通すること

♻️ サーキュラーエコノミー（循環型経済）を意識する

資源を無駄にせず、繰り返し使い続ける——それが「循環」の考え方です。

✓ ものをきれいに扱う

✓ 次に使う人のことを想像して消費する

✓ 修理して長く使う

✓ 適切な方法で手放す

企業や回収者にも、透明で持続可能な仕組み作りが求められています。
